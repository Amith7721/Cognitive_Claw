# 🧬 Cognitive Claw: Neural Agentic OS
**Redefining Human-AI Synergy for the Hackathon 2026**

**Cognitive Claw** is not just an AI assistant—it is a proactive **Personal Cognitive Operating System**. It evolves alongside the user by building a persistent **Cognitive DNA** profile, transforming raw behavioral data into a living, breathing digital partner.

---

## 🚩 The Problem
Traditional AI tools (ChatGPT, Claude, Gemini) are **reactive and generic**. 
*   They don't "know" you; every session starts from scratch.
*   They don't understand your work habits, peak focus hours, or learning style.
*   Users are forced to manually bridge the gap between their physical world (notes/tasks) and digital intelligence.

## 💡 The Solution: An Agentic OS
Cognitive Claw bridges this gap by implementing a **Neural Identity Layer**. It silently observes your tasks, research, and vision scans to calculate your "Cognitive DNA," allowing the AI to provide hyper-personalized briefs, schedule optimizations, and real-time behavioral insights. It doesn't wait for you to ask—it anticipates what you need.

---

## 🏛️ OpenClaw Architecture: The Agentic Core
Cognitive Claw is powered by **OpenClaw**, a modular agentic framework that treats AI personality and logic as "hot-swappable" DNA files.

*   **Soul Layer:** Defines the core ethical and behavioral persona of the agent.
*   **Memory Layer:** Provides the context of previous interactions and learned user patterns.
*   **Heartbeat Layer:** Manages real-time processing, notifications, and scheduled behavioral checks.
*   **Modular Skills:** A library of specialized Markdown files (Meeting Prep, Research, Productivity) that the agent "activates" based on current user needs.

---

## 🚀 Core Features

### 🧠 Neural Memory Vault (Cognitive DNA)
*   **Behavioral Learning:** Tracks productivity trends and research interests to build a "Genome."
*   **Efficiency Metric:** A real-time % score of your cognitive performance based on task completion.
*   **Aura Pulse:** A premium visual indicator of your current "behavioral health" (Radiant, Stable, High, or Critical).

### 👁️ Vision Synapse
*   **OCR Intelligence:** Scan handwritten notes, timetables, or complex charts.
*   **Agentic Extraction:** Automatically converts scanned images into Todoist tasks or Research insights.
*   **Digitization:** Effortlessly move physical knowledge into your persistent digital vault.

### 🎙️ Human-Fidelity Voice Assistant
*   **Neural TTS:** Uses "Studio" and "Enhanced" voices for a warm, natural, human-like interaction.
*   **Identity Context:** The assistant knows your name, your goals, and your learned patterns.
*   **Global Subtitles:** Real-time speech-to-text overlay for all AI interactions.

## 🛠️ Setup Instructions

### Prerequisites
*   Flutter SDK (Stable)
*   Android Studio / Xcode
*   An [OpenRouter API Key](https://openrouter.ai/)

### Installation
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/Amith7721/Cognitive_Claw.git
    cd cognitive_claw
    ```
2.  **Configuration:**
    Create a `.env` file in the root directory:
    ```env
    OPENROUTER_API_KEY=your_key_here
    ```
3.  **Dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run:**
    ```bash
    flutter run
    ```

---

## 📱 Usage Guide

1.  **Activation:** Complete the "System Activation" sequence to initialize the neural shell.
2.  **Neural Identity:** Fill out your "Neural Identity Profile" in Settings. The AI will immediately begin calculating your **Identity Tags**.
3.  **Vision Scans:** Use the "Synapse" feature to scan physical documents. Watch as the AI extracts tasks or simplifies complex data.
4.  **Evolve DNA:** Click the **"EVOLVE DNA"** button on the home dashboard to trigger a deep-learning analysis of your recent behavioral logs.
5.  **Daily Briefs:** Check the home screen every morning for a personalized brief synthesized specifically for your learned work patterns.
